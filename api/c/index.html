pinktrace - Pink's Tracing Library

Overview

pinktrace is a wrapper around ptrace(2) system call. It provides a robust API for tracing processes.

Attention:
This is a work in progress and the API is not stable.

Parts

This library is divided into two parts:

  • Core library (pinktrace)
  • Easy library (pinktrace-easy)
