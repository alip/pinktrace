bool pink_decode_socket_address ( pid_t  pid,
pink_bitness_t  bitness,
unsigned  ind,
long *  fd,
pink_socket_address_t *  paddr 
)

Get the socket address and place it in paddr.

Note:
This function decodes the socketcall(2) system call on some architectures.
If the address argument of the system call was NULL, this function returns true and sets addr->family to -1.
Parameters:
pidProcess ID of the child whose argument is to be received.
bitnessBitness of the child
indThe index of the argument. One of:
  • 1 (for connect, bind etc.)
  • 4 (for sendto)
fdThe pointer to store the socket file descriptor that resides in argument one with index zero. Set this to NULL if you don't need the file descriptor to be decoded.
paddrThe pointer to store the decoded socket address
Returns:
true on success, false on failure and sets errno accordingly.
Examples:
pink-simple-strace-freebsd.c, and pink-simple-strace-linux.c.
