void pink_easy_context_destroy ( pink_easy_context_t *  ctx)

Destroy a tracing context; destroys the process list and all the members of the list. The user data of the processes are free'd if a destructor function was provided with pink_easy_process_set_data(). The user data of the context is free'd if a destructor function was provided with pink_easy_context_new().
