Return the name of the given system call.

Note:
On ARM architecture, this function considers the system call architecture specific if scno is smaller than zero.
See also:
pink_util_get_syscall
Parameters:
scnoSystem call number
bitnessBitness of the child
Returns:
The name of the system call, NULL if system call name is unknown.
Examples:
pink-simple-strace-freebsd.c, and pink-simple-strace-linux.c.
