decode(...)

 

This function decodes the string at the argument of the given index.

Parameters:
  • pid - Process ID of the traced child
  • index - The index of the argument
  • maxlen - Max length of the string (Optional, defaults to -1, if smaller than zero, pinktrace tries to determine the string length itself)
  • bitness - The bitness of the traced child (Optional, defaults to pinktrace.bitness.DEFAULT_BITNESS)
Returns: str
The decoded string
Raises:
  • IndexError - Raised if the index is not smaller than MAX_INDEX
  • ValueError - Raised if the given bitness is either unsupported or invalid
  • OSError - Raised when the underlying ptrace(2) call fails.
