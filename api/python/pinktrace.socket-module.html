decode_address(...)

 

Decodes the socket address at the given argument index.

Parameters:
  • pid - Process ID of the traced child
  • index - The index of the argument
  • bitness - The bitness of the traced child (Optional, defaults to pinktrace.bitness.DEFAULT_BITNESS)
Returns: pinktrace.socket.Address
The decoded socket address
Raises:
  • IndexError - Raised if the index is not smaller than MAX_INDEX
  • ValueError - Raised if the given bitness is either unsupported or invalid
  • OSError - Raised when the underlying ptrace(2) call fails.

Note: This function decodes the socketcall(2) system call on some architectures.

has_socketcall(...)

 

Returns True if the socket calls - like connect, bind, sendto etc. - are implemented as subcalls of the socketcall(2) system call, False otherwise.

Parameters:
  • bitness - The bitness of the traced child (Optional, defaults to pinktrace.bitness.DEFAULT_BITNESS)
Raises:
  • ValueError - Raised if the given bitness is either unsupported or invalid

Note: Availability: Linux
